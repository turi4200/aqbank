# Projeto de teste AQBank

Feito em IOS.

### Instalação

1. Clone este repositório para o seu computador:

   ```bash
   git clone https://github.com/turi4200/aqbank.git
   ```

2. Acesse o diretório do projeto:

   ```bash
   cd aqbank
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Execute o projeto:

   ```bash
   npx react-native run-ios
   ```
